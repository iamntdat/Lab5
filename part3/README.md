# POS Tagging with Simple RNN

This project implements Lab 5: RNN for POS Tagging.

## Structure
- load_data.py — parse CoNLL-U
- dataset.py — PyTorch Dataset + padding
- model.py — Simple RNN model
- train.py — training + evaluation

## How to run
```
python train.py
```

## Report
- Implements Task 1–5 fully.
- Accuracy printed per epoch.
